SceneVision — AI Scene Classification

SceneVision adalah aplikasi web berbasis Flask + TensorFlow untuk mengklasifikasikan gambar pemandangan ke dalam beberapa kategori seperti:

🌲 Forest

🏔️ Mountain

🌊 Sea

🏙️ Buildings

🧊 Glacier

🛣️ Street

Fitur

Upload gambar

Prediksi kelas menggunakan model CNN

Tampilan modern dan responsif

Visualisasi dataset dan training

Monitoring hasil prediksi

Cara Menjalankan

1. Clone repository

git clone https://github.com/YGG0/TBPrakAI_SceneVision.git

2. Masuk folder project

cd TBPrakAI_SceneVision

3. Buat virtual environment

python -m venv .venv

.venv\Scripts\activate

4. Install dependency

pip install -r requirements.txt

5. Jalankan aplikasi

python app.py

6. Buka browser

http://127.0.0.1:5000
